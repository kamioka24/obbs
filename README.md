# obbs

## サイト概要
店舗オリジナル野球用品販売サイト(スマホ対応)

### サイトテーマ
お客様もお店側も笑顔になるサイト

### テーマを選んだ理由
前職で働いている時に、紙(手書き)で注文詳細やお客様情報の管理をしていて
大変だった経験があり、注文サイトがあれば効率よく物事が進むと思ったからです。

### ターゲットユーザ
顧客・遠方からお越しのお客様・新規ユーザーなど。
主に野球少年のお父さんお母さんや、中学、高校球児。

### 主な利用シーン
気づいた時にその場でスマホから注文可能
忙しくて来店できなくても配送先を指定して商品を受け取れる。
